# 🐾 貓爪截圖工具 Cat Paw Screenshot

一個可愛的 Chrome Extension 截圖工具，讓你輕鬆截圖並添加標註！

## ✨ 特色功能

- 🐱 **可愛懸浮貓爪按鈕** - 永遠浮在網頁最上層，可自由拖曳
- 📸 **一鍵截圖** - 點擊貓爪立即截取當前畫面
- 🎨 **簡單編輯工具** - 提供畫筆、方框、圓形、貓爪貼紙等工具
- 🌸 **療癒視覺風格** - 粉色系卡通風格，療癒你的心
- 💾 **本地處理** - 所有資料只在本地處理，不上傳雲端

## 🎯 編輯功能

1. ✏️ **畫筆** - 自由手繪標記
2. ⬜ **方框** - 框選重點區域
3. ⭕ **圓形** - 圓圈標記
4. 🐾 **貓爪貼紙** - 點一下蓋一個可愛貓爪印
5. ↩️ **復原** - 支援多步驟復原（最多 20 步）
6. 💾 **下載** - 儲存為 PNG 格式

## 📦 安裝方式

### 方法一：本地載入（開發模式）

1. 下載或複製此專案到本機
2. 開啟 Chrome 瀏覽器
3. 進入擴充功能管理頁面：
   - 在網址列輸入 `chrome://extensions/`
   - 或點擊右上角 ⋮ → 擴充功能 → 管理擴充功能
4. 開啟右上角的「開發人員模式」
5. 點擊「載入未封裝項目」
6. 選擇 `cat-paw-screenshot` 資料夾
7. 完成！貓爪按鈕會出現在所有網頁右下角

### 方法二：打包安裝

1. 在擴充功能管理頁面，點擊「封裝擴充功能」
2. 選擇 `cat-paw-screenshot` 資料夾
3. 產生 `.crx` 檔案
4. 拖曳 `.crx` 檔案到 Chrome 視窗安裝

## 🎮 使用方式

### 截圖

1. 在任何網頁上，你會看到右下角有一個粉色的貓爪按鈕
2. 點擊貓爪按鈕（會出現「啾～」動畫）
3. 自動截取當前畫面並開啟編輯頁面

### 編輯

1. 選擇工具（畫筆、方框、圓形、貓爪）
2. 在截圖上繪製標記
3. 可以調整顏色和粗細
4. 使用「復原」按鈕修正錯誤
5. 點擊「下載」儲存編輯後的圖片

### 拖曳貓爪

- 按住貓爪按鈕不放，拖曳到你喜歡的位置
- 鬆開滑鼠即可固定位置

## 📁 專案結構

```
cat-paw-screenshot/
├── manifest.json          # Extension 配置檔（Manifest V3）
├── background.js          # 背景腳本（處理截圖邏輯）
├── content.js             # 內容腳本（懸浮貓爪按鈕）
├── content.css            # 懸浮按鈕樣式
├── editor.html            # 編輯介面 HTML
├── editor.css             # 編輯介面樣式（可愛風格）
├── editor.js              # 編輯功能邏輯
├── assets/
│   ├── paw.svg           # 貓爪圖示
│   ├── cat.svg           # 小貓裝飾
│   └── icons/            # Extension 圖示（需自行添加）
└── README.md             # 說明文件
```

## 🔧 技術細節

- **Manifest Version**: V3
- **主要 API**:
  - `chrome.tabs.captureVisibleTab` - 截取可見分頁
  - `chrome.storage.local` - 本地儲存截圖資料
  - `chrome.runtime` - 訊息傳遞
- **Canvas API** - 圖片編輯功能
- **純 JavaScript** - 不依賴任何框架

## ⚠️ 注意事項

### Icon 圖示

專案中 `manifest.json` 需要的 icon 檔案（16x16, 48x48, 128x128）需要自行準備：

- 可以使用線上工具將 `assets/paw.svg` 轉換成不同尺寸的 PNG
- 或使用設計軟體製作
- 放置在 `assets/icons/` 資料夾中

### 權限說明

- `activeTab` - 截取當前分頁
- `tabs` - 取得分頁資訊
- `storage` - 儲存截圖資料
- `<all_urls>` - 在所有網頁注入貓爪按鈕

### 相容性

- 需要 Chrome 88 或更新版本
- 建議使用最新版本的 Chrome
